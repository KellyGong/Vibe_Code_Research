## What It Is

**Warp is not a traditional IDE.** Originally a modern terminal, Warp pivoted in mid-2025 to become the first **Agentic Development Environment (ADE)** â€” abandoning IDE-style coding entirely. The bet: the terminal's direct input/output architecture is better suited for agent-driven workflows than a code editor with file trees and tabs.

> "We think it should look more like a terminal than an IDE." â€” Zach Lloyd, Warp CEO

Developers become **orchestrators and intent-shapers**, not code typists.

## Credits & Pricing

Warp uses a **per-call credit system** â€” each AI request costs at least 1 credit. Actual cost scales with model, context size, and codebase scan depth.

| Plan | Price | Credits/month | Notes |
|------|-------|---------------|-------|
| **Free** | $0 | ~300 (new user bonus) | Core terminal features free forever; new users get free credits |
| **Build** | $20/mo | 1,500 | BYOK support; credits roll over 12 months |
| **Business** | $50/mo | 1,500 | SSO, ZDR, team management |

> âš ï¸ **Credit burn warning**: Heavy agentic sessions with many tool calls can exhaust credits fast. A single complex session with Opus 4.6 max + MCP tools can burn through dozens of credits. Use Auto (Cost-efficient) or BYOK to manage costs.
