## What It Is

**Antigravity** is Google's AI-native IDE, similar in concept to Cursor or Windsurf. It is a dedicated coding environment with AI built in, but with **fewer features** and a **restricted model set** compared to those tools.

## Features

- IDE-style editing with AI assistance
- Functionality is **similar to Cursor** but less mature — fewer agent features, no GPT

## Models

- **Claude** (Opus, Sonnet) and **Gemini** family only
- **No GPT** — You cannot use OpenAI models in Antigravity

## Credits & Pricing

- **Free for Google Pro subscribers** — If you have a Google Pro (or equivalent) subscription, Antigravity access is included
- **5-hour rolling credit refresh** — Credits renew on a rolling 5-hour window
- Credits are **limited**; heavy use will hit the cap

## Network: TUN Mode Required

Antigravity requires access to Google services. In regions where Google is blocked, you **must enable TUN mode** in your proxy/VPN software (e.g. Clash, Surge, Shadowrocket) before launching Antigravity — simple HTTP proxy is not enough; the IDE needs full system-level network routing via TUN.

## Quota Monitoring

Credits are limited and the built-in UI doesn't show detailed quota info. Use the community plugin **[AntigravityQuotaWatcher](https://github.com/wusimpl/AntigravityQuotaWatcher)** (2k+ stars) to monitor your usage in real time:

- **Status bar display** — Shows remaining quota for each model pool (Gemini, Claude, etc.) directly in the bottom bar
- **Color-coded alerts** — Green (≥50%), Yellow (30–50%), Red (<30%), Black (0%)
- **Dashboard panel** — Quota overview, connection status, weekly limit detection
- **Auto-detect** — Automatically detects Antigravity's local service port and auth info

Install from the Antigravity extension marketplace: search `AntigravityQuotaWatcher` by `wusimpl`.

## Pros

- **Free for Google Pro users** — No extra fee if you already subscribe
- **Integrated with Google ecosystem** — Fits if you're already in the Google stack

## Cons

- **Very limited features** — Fewer capabilities than Cursor/Windsurf
- **Requires TUN mode** — Must enable TUN in proxy software; simple HTTP proxy won't work
- **Limited credits** — 5-hour refresh is not generous for power users
- **Restricted model selection** — Claude + Gemini only; no GPT

## 简介

**Antigravity** 是 Google 的 AI 原生 IDE，定位类似 Cursor 或 Windsurf，是内置 AI 的编程环境，但 **功能较少**、**模型选择受限**。

## 功能

- 类 IDE 编辑 + AI 辅助
- 与 Cursor **类似但更简** — 智能体能力较少，且不支持 GPT

## 模型

- 仅支持 **Claude**（Opus、Sonnet）与 **Gemini** 系列
- **无 GPT** — 无法在 Antigravity 中使用 OpenAI 模型

## 额度与定价

- **Google Pro 用户免费** — 拥有 Google Pro（或等同）订阅即可使用 Antigravity
- **5 小时滚动刷新** — 额度按 5 小时滚动窗口刷新
- 额度 **有限**，重度使用容易触顶

## 网络：需开启 TUN 模式

Antigravity 需要访问 Google 服务。在 Google 被屏蔽的地区，启动 Antigravity 前 **必须在代理/VPN 软件（如 Clash、Surge、Shadowrocket）中开启 TUN 模式** — 普通 HTTP 代理不够，IDE 需要通过 TUN 实现系统级网络路由。

## 额度监控

额度有限，而内置 UI 不显示详细配额信息。可以使用社区插件 **[AntigravityQuotaWatcher](https://github.com/wusimpl/AntigravityQuotaWatcher)**（2k+ stars）实时监控用量：

- **状态栏显示** — 在底栏直接展示各模型池（Gemini、Claude 等）的剩余额度
- **颜色预警** — 绿色（≥50%）、黄色（30–50%）、红色（<30%）、黑色（0%）
- **Dashboard 面板** — 配额概览、连接状态、周限检测
- **自动检测** — 自动发现 Antigravity 本地服务端口和认证信息

安装方式：在 Antigravity 扩展市场搜索 `AntigravityQuotaWatcher`，认准作者 `wusimpl`。

## 优点

- **Google Pro 用户零额外成本** — 已有订阅即可用
- **与 Google 生态整合** — 适合已在 Google 体系内的用户

## 缺点

- **功能很有限** — 不如 Cursor/Windsurf 丰富
- **必须开 TUN 模式** — 代理软件需开启 TUN，普通 HTTP 代理不行
- **额度有限** — 5 小时刷新对重度用户不够
- **模型受限** — 仅 Claude + Gemini，无 GPT
